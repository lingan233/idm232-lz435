IDM232 - Final Case Study

The Overview

This project took an overall span of 10 weeks to complete as the final project of the class IDM 232.

Context and Challenge

Background:

  • Taken Web Design I, II, and Scripting for IDM I
  • Creating this project for the IDM 232 Scripting class
  • Want to do a team recommendation website for Genshin Impact
  • Researched on Dribbble and CodePen for inspiration

Description:

The website includes a home page, team detail page, recipe listing page, and recipe create page.

Timeline:

  • Week 2-4 working on Alpha design
  • Week 4-6 working on Alpha build
  • Week 6-10 working on Beta build
  • Week 10-12 working on the Final build

Purpose:

  • Class Project

The Problems:

  • Linking multiple tables in the database
  • Saving html form into database in multiple tables
  • Getting the value of html options using the database

Goals & Objectives:

  • A functional website using php and Mysql
  • Allow users to create and view team recipes

Process and Insight

Designs

  • Designed using Figma
  • Taken inspirations from Dribbble

Code/Dev

  • Starting small works well with coding. I have to erase many tables in the database and recreate some, so it was good that I did not input all the values in the first place.
  • Learning php is fun and painful at the same time.

The Solution
